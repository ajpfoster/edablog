What are the best practices associated with using classes vs. ids?

Sprint 2 - Technical Blog

Posted by Ash Foster on April 19, 2019

Hi team! Gonna have a go at explaining best practises for the css selectors class and id.

The general rule of thumb is if the element you are styling will be used more than once make it a class - classes are easier to handle and can always be overriden

This allows us the benefit of 2 things. 1 - You are able to still use the id selector for individual styling of specific elements and 2.

Class Inheritance

Class inheritance is an important feature of CSS (read programming) in that it allows us to pass attributes or styling to an identified class of elements INCLUDING multiple instances of that element AND child classes or sub classes of that element.
